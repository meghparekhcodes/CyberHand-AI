# 🖐️ CyberHand AI

An interactive browser-based hand tracking experience built using **MediaPipe Hands** and **p5.js**. CyberHand AI transforms real-time hand movements into a futuristic cyberpunk visualization with glowing hand skeletons, dynamic fingertip connections, and interactive particle effects.

---

## 🚀 Live Demo

🔗 **Try it here:**  
https://your-render-link.onrender.com

---

## ✨ Features

- 🖐️ Real-time hand tracking using MediaPipe Hands
- ✨ Neon glowing hand skeleton visualization
- 🧵 Dynamic "Cat's Cradle" fingertip connections
- 💥 Interactive particle effects on fingertip interaction
- 🎨 Cyberpunk-inspired UI and visual effects
- 📷 Live webcam integration
- 🌐 Runs entirely in the browser

## 💻 Run Locally

1. Clone this repository.

```bash
git clone https://github.com/yourusername/CyberHand-AI.git
```

2. Open the project folder.

3. Run the project using a local server (recommended).

Using Python:

```bash
python -m http.server
```

or use the **Live Server** extension in Visual Studio Code.

4. Open your browser and allow camera access.

5. Start interacting with CyberHand AI.
